# Blog
A Blog app written in Django

# Requirements
 * Python 3.8+
 * Django 3.2.8+
 * virtualenv
 * Mysql 8+
 
# Features
 * CRUD on Blogs
 * CRUD on Commens
 * CRUD on Tags
 * Can add multiple tags to a blog
 * User Creation
 * User Auth
 * Like & Dislike Blog
 * Like & Dislike Comment
 * Pin Comment
